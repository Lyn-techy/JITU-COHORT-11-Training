1. Will the below two functions output the same results? Explain your answer.


    Answer


    The first function foo1() is written correctly and will return an object {bar: "hello"} when called.
    The second function foo2() contains a typo in the return statement, the R is capitalized. Due to this,
    the JavaScript interpreter throws a syntax error when it encounters the capitalized Return keyword.
    The function will not execute and will not return any value. If one fixes the syntax error by changing
    the Return to return in the second function, it will output the same result as the first function.
    This further proves that JavaScript is case sensitive.

  3. Using a Prompt allow user to enter numbers separated by commas then print the sum of the
    numbers.


    Answer


    The function sumNumbers() uses the prompt() method to ask the user to enter numbers separated by commas.
    The input string is then split into an array of strings using the split() method, with "," as the separator.
    The map() method is used to convert the strings to numbers using the parseFloat() function.
    The reduce() method is used to sum the numbers in the array.
    Finally, the sum is printed to the console using console.log().
