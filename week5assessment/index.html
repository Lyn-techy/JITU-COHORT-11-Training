Question 1

What is the difference between state and props?

Solution
STATE PROPS
The data is passed within the component only. The data is passed from one component to another.
It is mutable(can be modified). It is immutable(cannot be modified).
State can be used only with the state components/class components. Props can be used with state and functional components.
State is both read and write. Props are read only.

Question 4

How are controlled and uncontrolled components different?

Solution
  1. In a controlled component, form data is handled by a React component. Whereas,
    in uncontrolled components, form data is handled by the DOM itself.
  2. Usage of component State is a must for controlled components. Use of state is completely
    optional for uncontrolled components, but one must use Refs in it.
  3. In controlled component, the form input elements values and mutations are totally driven by
    event handlers and value of the input element is always inferred from the state.
    In uncontrolled components, they don't use any states on input elements or any event handler.
